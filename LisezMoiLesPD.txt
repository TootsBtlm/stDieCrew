Git c'est la galère alors suivez les instructions :


Pour vous connectez à notre projet avec votre console :
	-créer un repertoir "git" pour tous vos projets
	-dans votre repertoire taper la commande :
	/git$ git clone https://github.com/TootsBtlm/stDieCrew.git
	-vous avez maintenant le répertoire du projet


Pour changer l'editeur de texte par défaut pour ne pas avoir l'editeur de merde dans la console :
	$ git config --global core.editor nomEditeur (gedit chez moi)

Pour télécharger/mettre à jour le projet sur votre ordi :
	/git/stDieCrew$ git pull
	-entrer les identifiants de votre compte Git

Pour charger/mettre à jour le projet pour tout le monde :
	/git/stDieCrew$ git -a commit
	-votre editeur de texte s'ouvre, vous devez note quelques mots sur les modif que vous venez de faire et votre nom. Ex : "MaJ de la classe BBQ.java par LePinSylvestre".
	-fermer en sauvegardant
	-taper:
	/git/stDieCrew$ git push
	-entrer les identifiants de votre compte Git
